# Ideation Tool

A prototype web application developed for a master thesis exploring how AI-generated suggestions influence designers’ decision-making, authorship, and creative confidence during ideation.

## What it does

Participants receive a design task and create an initial concept.

They enter their idea into the application.

The system returns three AI-generated suggestions:

1. Improvement suggestion  
2. Alternative direction  
3. Critical challenge

Participants reflect on the suggestions and decide whether to keep, modify, or rethink their concept.
